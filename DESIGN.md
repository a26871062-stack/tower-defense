# 塔防游戏 - 设计规范

## 概述

一款 Dungeon Crawler 像素风格的塔防游戏，玩家在迷宫路径旁建造箭塔、法师塔、炮塔抵御敌人波次。

---

## 视觉风格

### 像素艺术

- **精灵尺寸**：32x32 像素原始，游戏中 2x 缩放显示（64x64）
- **每帧步进**：2-4 像素，模拟经典 16-bit 游戏节奏感
- **去锯齿**：`texture_filter = Nearest`（Godot 最近邻采样），保留像素锐利边缘

|------|------|------|
| **深地牢** | 背景底色 | `#1E1A10` |

## 场景布局

### 游戏分辨率

- **画布**：1280x720（固定，不可缩放）
- **路径宽度**：40px（Line2D 渲染）
- **UI 锚点**：统一使用 `anchors_preset` 布局，避开刘海屏/异形屏

### 层级结构

```
Main (Node2D)
├── Path (Line2D)             — 路径绘制
├── PathVisual               — PathVisualizer 脚本
├── Enemies (Node2D)         — 敌人生成点
├── Towers (Node2D)            — 塔放置位置
├── Bullets (Node2D)          — 子弹容器
├── FloatingTexts (Node2D)    — 浮动伤害数字
└── UI (CanvasLayer)
    ├── TopBar (Panel)        — 顶栏：波次/金币/生命/等级
    ├── BuildPanel (Panel)     — 建造面板（右侧固定）
    ├── TowerUpgradeUI (Control) — 升级面板（屏幕居中）
    ├── StartWaveButton        — 开始波次按钮
    ├── NextLevelButton       — 进入下一关按钮
    └── MessageLabel (Label)  — 中央提示文字
```

## UI 组件规范

### TopBar（顶栏信息面板）

- **位置**：`offset_left=8, offset_top=8, offset_right=420, offset_bottom=210`
- **背景色**：`#38301E`，透明度 `0.5`，边框 `#8A6C30`
- **文字色**：`#F5EBD2`（羊皮纸色）
- **字号**：波次/等级 14px，金币/生命 20px

### BuildPanel（建造面板）

- **位置**：屏幕右侧固定，`offset_right=-20`
- **按钮**：竖排 3 个塔按钮，每个带 `CostLabel`
- **可用状态**：`bg_color = #594830`，`border_color = #D4B42C`
- **不可用（钱不够）**：半透明 `modulate.a = 0.5`

### TowerUpgradeUI（升级面板）

- **位置**：屏幕居中，`anchors_preset = 8`（居中），尺寸 `240x260`
- **背景**：`#38301E`，边框 `#D4B42C`
- **结构**：
  ```
  Panel
  └── VBox
      ├── TowerNameLabel (塔名，金色 20px)
      ├── UpgradeTowerLevelLabel (等级 14px)
      ├── UpgradeButton (升级，金色文字)
      │   └── CostLabel (花费)
      └── SellButton (出售，红字)
          └── ValueLabel (返还额)
  ```
- **mouse_filter**：`Control = PASS`，`Panel/VBox = IGNORE`

### 开始/下一关按钮

- **文字**：`#D4B42C` 金色，16px
- **按下态**：`#8A6C30` 深一号的棕色

## 游戏数值规范

### 经济系统

```
初始金币：100
初始生命：20
箭塔造价：50
法师塔造价：80
炮塔造价：100

塔升级费用：
  Lv1→Lv2：塔造价 × 0.8
  Lv2→Lv3：塔造价 × 1.2

出售返还：投入 × 50%（向下取整）

每关奖励：第2关 +50金币 / 第3关 +80金币 / 第4关 +5生命 / 第5关 +100金币
```

### 战斗参数

```
箭塔：攻击范围 150px，伤害 25，攻速 1.0/s
法师塔：攻击范围 130px，伤害 35，攻速 0.7/s（溅射）
炮塔：攻击范围 120px，伤害 60，攻速 0.4/s（范围伤害）

敌人基础属性（随波次倍率递增）：
  第1波 mult=1.0 → 第5波 mult=2.0
  HP = 基础HP × mult
  速度 × (1.0 + (wave-1) × 0.05)
```
